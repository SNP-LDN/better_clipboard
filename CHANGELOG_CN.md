# Better Clipboard 更新日志

**语言：** 中文 | [English](CHANGELOG_EN.md)

本文件记录 Better Clipboard 的重要功能与界面变化。

## 未发布

更新时间：2026-07-20

### 新增

- 新增浅色、深色和跟随 Windows 系统三种明暗模式。
- 新增标准与柔和玻璃两套界面风格。
- 选择玻璃风格后，可通过滑块将透明度调节为 55% 至 95%，并实时预览效果。
- 历史记录新增复选框，可同时选择多条记录并通过“删除所选”批量删除。
- 批量删除按钮会显示已选数量，并在删除前请求确认。
- 加入 Better Clipboard 品牌 Logo，并应用于窗口、标题栏、系统托盘和可执行文件图标。
- 应用小图标的剪贴板纸面改为白色，同时保持外部透明，以提高深色背景下的辨识度。

### 改进

- 现在可以按住标题栏拖动剪贴板历史窗口。
- 降低选中项、标签页和下拉选项的高亮亮度，使深色模式下的视觉反馈更柔和。
- 加宽记录操作区并固定按钮尺寸，避免收藏和删除按钮被压缩。
- 调整收藏与删除按钮的图标、间距和悬浮提示，提高可读性和操作准确性。
- 优化下拉框、标签页以及不同主题下的悬停、选中和禁用状态。
- 主题和玻璃透明度设置可在历史窗口与独立设置窗口中同步使用。

### 修复

- 修复收藏和删除按钮在较窄窗口中被挤压、文字或图标难以辨认的问题。
- 修复在复选框或操作按钮上点击时可能触发记录打开或粘贴的问题。
- 修复列表刷新后当前记录选择状态容易丢失的问题。
- 修复应用在深色标题栏和 Windows 托盘中使用默认或低对比度图标的问题。
