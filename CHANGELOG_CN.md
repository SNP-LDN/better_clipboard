# Better Clipboard 更新日志

**语言：** 中文 | [English](CHANGELOG_EN.md)

本文件记录 Better Clipboard 的重要功能与界面变化。

## 1.0.4 - 2026-07-22

### 修复

- 修复历史窗口失去焦点时重复执行关闭流程，导致应用在后台无提示崩溃的问题。
- 修复系统托盘右键菜单左侧显示成排空白图标方块的问题。
- 未处理的界面异常现在会显示错误报告提示，并在诊断日志中保存完整异常信息和调用位置。
- 诊断日志现在会记录应用的正常退出及退出代码，便于区分主动退出与异常终止。

## 1.0.3 - 2026-07-22

### 新增

- 每个已安装版本首次成功启动后显示明确的安装或更新完成提示，包括后台运行状态、开机启动状态和安装位置。

### 修复

- 修复原生系统托盘 API 入口名称错误导致安装后程序无法正常启动的问题。
- 系统托盘初始化失败时不再使整个应用退出，剪贴板监听和全局快捷键仍可继续工作。

## 1.0.2 - 2026-07-22

更新时间：2026-07-22

### 新增

- 在剪贴板历史窗口底部新增实时私有内存占用显示，每两秒刷新，并在窗口关闭时释放监控资源。
- 集成 Velopack 安装与在线更新，可通过 GitHub Releases 检查、下载并安装新版本。
- 设置窗口新增当前版本显示和手动“检查更新”入口。
- 安装程序默认创建桌面、开始菜单和开机启动快捷方式。
- 新增浅色、深色和跟随 Windows 系统三种明暗模式。
- 新增标准与柔和玻璃两套界面风格。
- 选择玻璃风格后，可通过滑块将透明度调节为 55% 至 95%，并实时预览效果。
- 历史记录新增复选框，可同时选择多条记录并通过“删除所选”批量删除。
- 批量删除按钮会显示已选数量，并在删除前请求确认。
- 加入 Better Clipboard 品牌 Logo，并应用于窗口、标题栏、系统托盘和可执行文件图标。
- 应用小图标的剪贴板纸面改为白色，同时保持外部透明，以提高深色背景下的辨识度。

### 改进

- 使用原生 Windows 托盘接口替代 WinForms 托盘组件，降低后台常驻内存与程序集加载数量。
- 历史窗口关闭后会释放控件树、列表、计时器事件和图片缩略图，不再长期隐藏并持有界面资源。
- 图片缩略图改为可见时按需加载，并使用有数量上限的 LRU 缓存，避免历史图片持续累积内存。
- 现在可以按住标题栏拖动剪贴板历史窗口。
- 降低选中项、标签页和下拉选项的高亮亮度，使深色模式下的视觉反馈更柔和。
- 加宽记录操作区并固定按钮尺寸，避免收藏和删除按钮被压缩。
- 调整收藏与删除按钮的图标、间距和悬浮提示，提高可读性和操作准确性。
- 优化下拉框、标签页以及不同主题下的悬停、选中和禁用状态。
- 主题和玻璃透明度设置可在历史窗口与独立设置窗口中同步使用。

### 修复

- 修复收藏和删除按钮在较窄窗口中被挤压、文字或图标难以辨认的问题。
- 修复在复选框或操作按钮上点击时可能触发记录打开或粘贴的问题。
- 修复列表刷新后当前记录选择状态容易丢失的问题。
- 修复应用在深色标题栏和 Windows 托盘中使用默认或低对比度图标的问题。
